# 将棋AI分析サービス

やねうら王を統合した将棋AI分析Webサービス

---

## 🚀 デプロイ方法

### 1. GitHubリポジトリ作成済み
このファイル一式をGitHubリポジトリにアップロード

### 2. Render.comでデプロイ

1. https://render.com/ にアクセス
2. GitHubアカウントでログイン
3. 「New +」→「Web Service」
4. リポジトリを選択
5. 設定：
   - **Environment**: `Docker`
   - **Plan**: `Free`
6. 「Create Web Service」をクリック

### 3. 完了！

15〜20分でビルド完了後、URLが発行されます。
